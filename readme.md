## What is Kalandra?
Kalandra is a limited open-source low-level programming language (inspired by C) that targets processors such as the Intel 8088.

## Compiling Kalandra from Source:
There's a build script called `run.sh`, it will automatically check (and ask to download) dependencies, compile the compiler and install it for you.

### To run `run.sh`:
Change directory to this folder in a linux/unix terminal.
```bash
cd /here/
```
Make `run.sh` executable, if not already.
```bash
chmod +x run.sh
```
Execute `run.sh`.
```bash
./run.sh
```
