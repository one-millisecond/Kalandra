## What is Kalandra?  
Kalandra is a lightweight, fully modular kernel designed as a drop-in replacement for MS-DOS on 8086/8088 IBM PC-compatible systems with BIOS support.
### Features (Current & In-Dev):
- Preemptive Multitasking
- KalandraAPI
